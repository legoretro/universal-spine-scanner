# Create The Private GitHub Repo

GitHub repo name:

```text
universal-spine-scanner
```

Use these settings:

- Visibility: Private
- README: Do not add
- `.gitignore`: Do not add
- License: Do not add

After the empty repo exists, run these commands from this folder:

```bash
git remote add origin https://github.com/YOUR-GITHUB-USERNAME/universal-spine-scanner.git
git push -u origin main
```

Do not upload `.env`.

The repo is only for the scanner app. The local eBay listing studio is not included.
